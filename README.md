# Tic-Tac-Toe-
**Tic-Tac-Toe Game – Description**  Tic-Tac-Toe is a classic two-player strategy game played on a 3×3 grid. The game is simple, engaging, and easy to understand, making it popular among players of all ages. One player uses **“X”** and the other uses **“O”**. Players take turns marking an empty cell in the grid with their respective symbol.
